# JUN WEI — AI 科技個人品牌量化交易網站

本專案為 **JUN WEI** 的高質感個人品牌網站，專為展示全職交易員的量化與 AI 交易系統整合願景所設計。網站融合現代極簡與科技美學，採用 **白色、米色、淺灰色** 的高端配色，搭配玻璃擬態（Glassmorphism）與科技感資訊流（HUD）介面，提供流暢的動態粒子背景、作品集燈箱（Lightbox）預覽以及一個即時互動的「模擬交易系統（Trading Lab）」。

---

## 🚀 核心功能與特色

### 1. 視覺設計與互動效果
* **極簡科技美學**：使用 Apple、Humane 風格的白、米、淺灰高質感配色，告別傳統科技網站一成不變的暗色系，呈現明亮優雅的太空艙控制面板感。
* **AI 粒子連線背景**：使用 HTML5 Canvas 繪製流動粒子，粒子之間會隨距離自動連結成網絡；並會隨滑鼠移動產生互動排斥，連線至游標位置。
* **HUD 科技感介面**：首頁全螢幕 Hero 區域以視覺、粒子、AI 數據流與動態座標展示，無冗餘自我介紹文字，展現極高純度的品牌形象。
* **玻璃擬態卡片**：使用 `backdrop-filter` 與精緻的半透明白邊框、柔和陰影，營造磨砂玻璃質感。

### 2. Portfolio 作品集展示區
* **自適應格狀版面**：桌面版採用 5 卡片非對稱矩陣排列，手機版自動切換為優雅的單欄模式。
* **Lightbox 燈箱預覽**：點擊任何作品即可開啟全螢幕高解析度燈箱預覽，支援 ESC、點擊背景或按鈕關閉。
* **無縫失效容錯**：若圖片檔案不存在，系統會**自動動態渲染一個帶有科技圖表與專案編號的高質感 SVG 預覽圖**，確保網頁無損顯示、絕不報錯。

### 3. About Me 個人介紹
* 採用玻璃擬態卡片，清楚列出 4-5 年實務交易經驗，並詳細描述未來結合程式設計與數據分析的四大 AI 整合願景。
* 右側附帶有 **動態雷達掃描動畫（Radar Active）** 的診斷數值面板，提升科技互動感。

### 4. Trading Lab 模擬交易實驗室
* **即時價格隨機步隨（Random Walk）**：點擊 **Start Simulation** 後，模擬市場價格每秒更新並隨機波動。
* **動態均線計算**：即時計算並在專屬 Canvas 圖表上繪製 5 週期均線 (MA5) 與 20 週期均線 (MA20) 曲線。
* **均線交叉訊號 (MA Cross)**：
  * 當 **MA5 > MA20**，顯示綠色發光閃爍的 **BUY** 訊號。
  * 當 **MA5 < MA20**，顯示紅色發光閃爍的 **SELL** 訊號。
* **完全本地端執行**：所有模擬均在瀏覽器端使用 Vanilla JS 完成，無需連接任何後端 API。

## 🌐 GitHub Pages 部署教學

本網站為純靜態網站，可以直接使用 GitHub 提供的免費靜態網頁託管服務進行部署。請按照以下步驟操作：

### 步驟 1：建立並推送至 GitHub 倉庫
1. 登入您的 GitHub 帳號，並建立一個名為 `introduce-yourself` 的新公開倉庫 (Public Repository)。
2. 在您的本機專案目錄中，初始化 Git 並關聯該遠端倉庫：
   ```bash
   git init
   git add .
   git commit -m "feat: init personal branding website"
   git branch -M main
   git remote add origin https://github.com/helear2252-ctrl/introduce-yourself.git
   git push -u origin main
   ```

### 步驟 2：開啟 GitHub Pages 功能
1. 前往您在 GitHub 上的該項目倉庫頁面。
2. 點擊頂部選單的 **Settings** (設定)。
3. 在左側導覽列中，找到 **Code and automation** 區塊，點擊 **Pages**。
4. 在 **Build and deployment** 下方的 **Source** 下拉選單中，選擇 **Deploy from a branch**。
5. 在下方 **Branch** 下拉選單中，選擇 `main` 分支，並保持目錄為 `/ (root)`。
6. 點擊 **Save**。

### 步驟 3：訪問您的網站
1. 儲存設定後，GitHub 會啟動自動化部署工作流程 (GitHub Actions)。
2. 約等待 1 至 2 分鐘，重新整理 Pages 設定頁面。
3. 頁面頂端將會顯示您的專屬網站網址，格式通常為：
   `https://<你的GitHub帳號>.github.io/introduce-yourself/`
4. 點擊該連結，即可看到您已成功部署的 AI 科技個人網站！
